# makefile_example
*makefile example for simple C projects*

The project is organized like this:
~~~
.
├── inc
│   ├── foo.h
│   ├── func.h
│   └── main.h
├── lib
├── main
├── main.o
├── makefile
├── obj
│   ├── foo.o
│   ├── func.o
│   └── main.o
└── src
    ├── foo.c
    ├── func.c
    └── main.c
~~~
